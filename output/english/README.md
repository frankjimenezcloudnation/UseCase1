# WTP component analysis — documentation package

Date: 2026-07-20 · Use Case 1 · test fund SPF

This package contains two folders:
- **human-readable/** — documentation in plain language (for experts/stakeholders). Start with `README.md` or `full-document.md`. Also contains `4-review-list.md` with all concrete review points.
- **for-ai/** — the technical/structured documents (for AI agents and developers): `project-context.md`, `golden-testset.md`, `deliverables-and-test-scenarios.md`, `final-result.md`.

Both folders contain the complete deliverables and the test scenarios linked to them.

Not in this package (located in the project repository): the raw results (`analyse-resultaat.json`), the test set as `.json`, the anonymised source files (`.docx`) and the backend code.
